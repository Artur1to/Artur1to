# Как установить новую версию профиля

## Самый надёжный способ

1. Распакуй архив `Artur1to-profile-v2.zip`.
2. Открой репозиторий `Artur1to/Artur1to`.
3. Удали старый `README.md`.
4. Нажми **Add file → Upload files**.
5. Выдели **все файлы** из распакованной папки:
   - `README.md`
   - `artur-header.gif`
   - `section-about.png`
   - `section-stack.png`
   - `section-projects.png`
   - `section-interests.png`
   - `artur-character.png`
   - `stack-board.png`
   - `project-alttools.png`
   - `project-altmanga.png`
   - `interests-board.png`
   - `artur-footer.png`
6. Нажми **Commit changes**.
7. Обнови страницу профиля.

Все файлы должны лежать в корне репозитория рядом с `README.md`.

## Важно

- ZIP-файл в репозиторий загружать не нужно.
- Старые изображения можно удалить после того, как новая версия заработает.
- Новый `README.md` — обычный текстовый файл небольшого размера.
